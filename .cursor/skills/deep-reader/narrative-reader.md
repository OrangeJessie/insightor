# Narrative Reader — 叙事类深度阅读子技能

适用：小说、传记、纪实文学、回忆录、史诗等以**人物 + 情节 + 时间线**驱动的文本。

---

## L1: 章回/场景穷举索引 — `L1-scenes.md`

**覆盖率 = 100%**。每一章/每一个逻辑场景都必须有条目，无一遗漏。

### 格式

```markdown
# L1 — 章回/场景索引

> 共 {N} 章/回, {M} 个场景

---

## Ch.{N} {回目/章名}

### 场景 {N}.1: {场景简名}
- **地点**: {空间}
- **人物**: {出场人物, 用 ID 标注}
- **事件**: {1-2 句概括}
- **关键意象**: {出现的重要意象/物品}
- **情绪基调**: {如：欢乐、压抑、讽刺}
- **话语层** *(有潜台词/反讽/多层含义时必填)*:
  - 📝 "{关键台词原文}"  — *{说话人}*
  - 表层: {字面意思}
  - 深层: {真实意图/人物心理/社交策略}
  - 效果: {这句话对情节/人物关系/读者理解产生什么影响}
- **名场面标签**: #{关键词标签1} #{关键词标签2}
- **→ L2**: {关联的 L2 报告和段落}
- **→ L3**: {关联的 L3 洞察文件, 如有}

### 场景 {N}.2: {场景简名}
- ...
```

### 场景划分标准

一个「场景」= 一段连续的时空中，人物组合基本不变、围绕单一事件展开的叙述片段。场景切换的标志：
- 地点变化
- 时间跳跃
- 核心人物组合改变
- 叙事焦点转移

### 示例（红楼梦第 27 回）

```markdown
## Ch.27 滴翠亭杨妃戏彩蝶 埋香冢飞燕泣残红

### 场景 27.1: 宝钗扑蝶
- **地点**: 大观园（滴翠亭附近花丛）
- **人物**: 薛宝钗(C03)
- **事件**: 宝钗在花丛中追逐一对玉色蝴蝶，一路扑到滴翠亭外
- **关键意象**: 蝴蝶（玉色）、团扇、花丛
- **情绪基调**: 天真烂漫（表层）
- **名场面标签**: #宝钗扑蝶 #戏彩蝶 #蝴蝶
- **→ L2**: reports/02-symbols.md§蝴蝶, reports/01-entities.md§C03
- **→ L3**: insights/insight-baochai-mask.md

### 场景 27.2: 滴翠亭偷听
- **地点**: 滴翠亭
- **人物**: 薛宝钗(C03), 小红(C28), 坠儿(C29)
- **事件**: 宝钗无意间听到小红和坠儿的私密对话，使「金蝉脱壳」之计嫁祸黛玉
- **关键意象**: 亭、帘（窥探/遮蔽）
- **情绪基调**: 紧张、精明
- **话语层**:
  - 📝 "颦儿，我看你往那里藏！" — *宝钗*
  - 表层: 和黛玉捉迷藏的玩笑
  - 深层: 故意制造「黛玉在此偷听」的假象，将嫌疑转嫁给黛玉，保护自己不被小红怀疑
  - 效果: 一句话完成自保+嫁祸，暴露宝钗本能级的社交算计能力
- **名场面标签**: #滴翠亭偷听 #金蝉脱壳 #嫁祸黛玉
- **→ L2**: reports/04-rules.md§行为边界, reports/05-themes.md§真与伪

### 场景 27.3: 黛玉葬花
- **地点**: 花冢
- **人物**: 林黛玉(C01)
- **事件**: 黛玉独自在花冢前葬花，吟《葬花吟》
- **关键意象**: 落花、花锄、花囊、流水
- **情绪基调**: 悲凉、自伤
- **名场面标签**: #黛玉葬花 #葬花吟 #落花
- **→ L2**: reports/02-symbols.md§花, reports/06-style.md§诗词
- **→ L3**: insights/insight-daiyu-flower-burial.md
```

### 话语层示例（红楼梦第 49 回 — 湘云评宝琴披风）

```markdown
### 场景 49.3: 众人赏雪赏宝琴
- **地点**: 大观园（雪中）
- **人物**: 史湘云(C06), 薛宝琴(C30), 众姐妹
- **事件**: 宝琴披着贾母赐的凫靥裘出场，众人惊艳
- **关键意象**: 雪、凫靥裘（野鸭毛披风）
- **情绪基调**: 欢快 + 微妙的社交试探
- **话语层**:
  - 📝 "这一件衣裳也只配她穿，别人穿了，实在不配。——那是孔雀毛的吧？……哎呀，是野鸭子头上的毛！可见老太太疼你了。这样疼宝玉，也没见给宝玉一件。" — *湘云*
  - 表层: ① 先夸（只配她穿）→ ② 认错材质（以为孔雀毛，其实是野鸭子毛）→ ③ 转赞（贾母很疼你）→ ④ 抬高（连宝玉都没有）
  - 深层:
    - **口快本色**: "野鸭子毛"脱口而出，从"孔雀"降格为"野鸭子"，无意间降低了披风档次——这是湘云心直口快的典型表现
    - **即时补救**: 意识到说了不太得体的话，立刻用「贾母疼你」「宝玉都没有」来拉升，变成了超高规格的赞美——显示她虽然口快但并非没有情商
    - **无意识嫉妒?**: "宝玉都没有"这句暴露了她下意识的参照系——湘云自己也是贾母养大的，宝琴初来就获此待遇，这里可能有一丝不自知的比较心理
  - 效果: 一句话里同时展现了湘云的率真、情商、和微妙的心理波动——是曹雪芹刻画人物语言最精妙的段落之一
- **名场面标签**: #湘云 #宝琴 #凫靥裘 #野鸭子毛 #贾母疼宝琴
- **→ L2**: reports/06-style.md§话语分析, reports/01-entities.md§C06
- **→ L3**: insights/insight-xiangyu-speech-style.md
```

### 话语层触发条件

**不是每个场景都需要话语层**。以下情况必须标注：

| 条件 | 示例 |
|------|------|
| 台词的表层意思 ≠ 真实意图 | 宝钗"颦儿我看你往那里藏"（嫁祸） |
| 一句话里有**自我矛盾/转折** | 湘云"野鸭子毛……可见老太太疼你"（贬→褒） |
| 台词暴露了说话人**未意识到的心理** | 湘云无意识的比较心理 |
| 存在**反讽**（作者让人物说出与事实相反的话） | 王熙凤的客套话（表面热情实则算计） |
| 台词是**权力博弈**的载体 | 王夫人貌似温和的"建议"实为命令 |
| 台词的含义只有**读者能解码**（dramatic irony） | 秦可卿托梦中的暗示 |

不触发的情况：叙述性对话（传达信息）、日常闲聊（无潜台词）。

### 名场面标签规则

- 每个场景 **至少 2 个标签**
- 标签应覆盖：场景通俗名称、核心人物行为、关键意象
- 包含**群众认知中的俗称**（如 #宝钗扑蝶 而非仅 #杨妃戏彩蝶）
- 标签用于 Grep 命中，尽量使用用户可能搜索的词汇

**核心关系字段说明（每个人物必须填写）：**

核心关系包含两部分：**关系概述**（当前/总结性态度）和 **关系弧线**（动态演变过程）。

#### 第一部分：关系概述

回答"A 对 B 总体上是什么态度"。每条关系 4 要素：

```
- {对方ID}({名字}) — {关系标签} | {态度色标} {态度关键词} | {态度说明} | {关键证据场景}
```

态度色标：
- 💚 正面（喜爱、信任、亲密）
- 🟡 复杂/矛盾（表面一套内心一套、态度有转变）
- 🔴 负面（厌恶、排斥、对立）
- ⚪ 中性/功能性（上下级、无情感倾向）

#### 第二部分：关系弧线（动态关系必须填写）

回答"A 和 B 的关系是怎么变化的"。用阶段+转折点建模关系的时间演变。

**判断标准**：如果 A 对 B 的态度在全书中发生了明显变化（如从敌对到友好、从信任到疏远），则必须展开关系弧线。如果始终稳定（如紫鹃对黛玉的忠诚从未动摇），则只写关系概述即可。

**格式**：

```markdown
- **关系弧线 → {对方}**:
  - **Phase 1** (Ch.X—Ch.Y): {态度色标} {态度} — {行为表现} | 证据: {场景}
  - 🔄 **转折**: {事件名} (Ch.Z) — 驱动: {内因/外因说明}
  - **Phase 2** (Ch.Z—Ch.W): {态度色标} {态度} — {行为表现} | 证据: {场景}
  - 🔄 **转折**: ...
  - **Phase 3** ...
  - ❓ **未解**: {关于这段关系未明确的争议点}
```

**转折点的三类驱动因素**：

| 标记 | 驱动类型 | 说明 | 示例 |
|------|---------|------|------|
| 🧠 | **认知转变** | 一方对另一方的理解/看法发生了根本改变 | 黛玉发现宝钗并非敌人 |
| ⚡ | **外部事件** | 外部事件迫使关系改变 | 抄检大观园导致贾母被架空 |
| 📉 | **权力/结构变化** | 双方的地位/权力格局发生了偏移 | 王夫人势力上升，贾母话语权下降 |

**重要**：态度和转折都必须基于原文证据。有争议的标注 ❓ 并简述争议。

#### 哪些关系需要展开弧线？

| 条件 | 是否展开 | 示例 |
|------|---------|------|
| 态度发生了至少一次明确转变 | ✅ 展开 | 黛玉→宝钗（敌意→和解） |
| 权力/地位关系发生偏移 | ✅ 展开 | 贾母→王夫人（掌权→被架空） |
| 存在学界/读者争议的态度 | ✅ 展开 | 贾母→宝钗（真心还是客气？） |
| 全书始终稳定不变 | ❌ 只写概述 | 紫鹃→黛玉（始终忠诚） |
| 互动极少，关系功能性 | ❌ 只写概述 | 小红→宝钗（无直接互动） |

主要角色之间的关系**优先展开弧线**。次要角色之间只有在关系变化对情节有显著影响时才展开。

**叙事功能字段说明（每个人物必须填写）：**

叙事功能用于回答"这个角色存在的意义是什么"。每个人物至少标注 1-3 项：

| 标记 | 功能类型 | 说明 | 示例 |
|------|---------|------|------|
| 🔧 | **情节触发器** | 该角色的行为直接触发了某个关键事件 | 小红的私情 → 触发滴翠亭偷听 |
| 🪞 | **镜像对照** | 与另一角色构成映射/反面/变体 | 小红(务实) ↔ 晴雯(理想) |
| 📐 | **主题载体** | 承载/体现了某个核心主题 | 小红 → 阶级流动的可能性与代价 |
| 🔗 | **结构补充** | 为叙事结构提供平行线/副线/呼应 | 小红+贾芸 → 平行于宝黛的底层爱情线 |
| 🎭 | **氛围/功能** | 提供喜剧、缓冲、旁观视角等功能 | 刘姥姥 → 外部视角观照贾府 |

**主要角色和次要角色都必须填写叙事功能。** 次要角色尤其重要——读者往往能记住名场面但不理解次要角色的文学设计意图。

#### 更多角色示例

```markdown
### C05 — 贾母 / 史太君
- **身份**: 贾府最高权威，荣国公之妻，宝玉祖母，黛玉外祖母
- **核心关系**:
  - C02(宝玉) — 孙子 | 💚 溺爱 | 最宠爱的孙辈，多次庇护 | Ch.33
  - C01(黛玉) — 外孙女 | 💚 疼爱 | 因亡女贾敏之故格外疼惜 | Ch.3, Ch.97
  - C03(宝钗) — 亲戚晚辈 | 🟡 欣赏但有距离 | 表面夸赞稳重，实际审美不合 | Ch.22, Ch.35, Ch.40
  - 王夫人 — 儿媳 | 🟡 信任→被架空 | 前期和谐，后期权力被转移 | Ch.74, Ch.97
- **关系弧线 → C03(宝钗)**:
  - **Phase 1** (Ch.4—Ch.35): 💚 礼节性欣赏 — 欣赏宝钗稳重懂事，出资为其过生日 | 证据: Ch.22 亲自操办生日, Ch.35 "从我们家四个女孩儿算起，都不如宝丫头"
  - 🔄 **转折**: 蘅芜苑评价 (Ch.40) — 驱动: 🧠 审美/价值观冲突——看到宝钗房间过于素净，直言"年轻姑娘房里这样素净，忌讳"，暴露出深层审美不合
  - **Phase 2** (Ch.40—Ch.84): 🟡 表面不变/实际疏远 — 公开场合仍赞宝钗，但在宝玉婚事上始终未明确表态支持金玉良缘 | 证据: Ch.54 暗批"才子佳人"套路(有学者认为指向金玉), Ch.66 贾母笑言"不是冤家不聚头"(暗指宝黛)
```
